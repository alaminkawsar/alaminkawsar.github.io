Research Experience

Digital Biomarker Modeling for Dementia

This research project conducted on digital biomarker modeling for the early detection of Mild Cognitive Impairment (MCI) and Alzheimer’s disease using multimodal data collected from mobile and wearable devices. The work focused on extracting behavioral and physiological signals (e.g., activity patterns, movement variability, and temporal features) as potential digital biomarkers. I performed extensive data preprocessing and exploratory analysis using Pandas and Kotlin DataFrame, applied statistical analysis to identify discriminative features, and developed classification models to distinguish healthy subjects from MCI and Alzheimer’s patients. The research emphasized data quality, feature engineering, and evaluation of model performance to assess the feasibility of mobile and wearable sensing as scalable tools for cognitive health monitoring.

Automated Test Case Creation from UI Guide using Gen AI

This research project, conducted at Samsung R&D Institute, investigates the efficacy of Large Language Models (LLMs), Vision-Language Models (VLMs), and Computer Vision Models in generating test cases from UI guides. We developed a framework that employs a robust method for detecting functional flows within software from UI guides using Vision-Language Models (VLMs), Large Language Models (LLMs), and Computer Vision models, converting these functional flows into test cases through an algorithmic approach. Additionally, we generated expected outcomes for specific functional flows using VLMs. My contributions included conducting few-shot prompting experiments and fine-tuning VLMs (such as CogVLM and Idefics2) for UI screen identification and functional flow mapping. I also performed UI screen matching with their corresponding descriptions using the LLaMa-3-8B model. Our framework effectively generates test cases from extensive UI guides with minimal error and within a reasonable time frame.

Hand Gesture Recognition

  • Conducted extensive research on 18 TensorFlow-based deep learning models for hand gesture recognition with a focus on mobile and edge deployment.
    • Lightweight & Mobile-Optimized Models (Real-time, Edge Devices): SSD MobileNet V1, SSD MobileNet V2, SSD MobileNet V2 FPNLite, SSD MobileNet V3 Small, SSD MobileNet V3 Large, CenterNet MobileNetV2
    • Balanced Models (Speed vs Accuracy): SSD ResNet50 V1 FPN, SSD ResNet101 V1 FPN, CenterNet ResNet50, CenterNet HourGlass104 (Reduced Input)
    • High-Accuracy Models (Heavy / Research-Oriented): Faster R-CNN ResNet50, Faster R-CNN ResNet50 FPN, Faster R-CNN ResNet101, Faster R-CNN ResNet101 FPN, Faster R-CNN Inception ResNet V2
    • Anchor-Free & Modern Architectures: CenterNet HourGlass104, CenterNet ResNet101
    • EfficientDet Family (Scalable Detection Models): EfficientDet D0, EfficientDet D1, EfficientDet D2, EfficientDet D3, EfficientDet D4, EfficientDet D5, EfficientDet D6, EfficientDet D7
  • Evaluated all models using the HaGRID(Hand Gesture Recognition Dataset), consisting of 19 gesture classes, each approximately 36GB in size.
  • Due to the large-scale nature of the dataset, randomly sampled 5,000 images per class to create a balanced and computationally manageable evaluation set.
  • Preprocessed and resized the dataset into multiple input resolutions: 244×244, 320×320, 480×480, and 640×640, enabling analysis of input-size impact on model performance.
  • Benchmarked each model across different resolutions to compare accuracy, inference latency, computational cost, and memory footprint.
  • Applied post-training quantization using fixed bit-width configurations to optimize models for mobile devices.
  • Converted optimized models to TensorFlow Lite (TFLite) format for on-device inference testing.
  • Analyzed performance degradation ratio by comparing FP32 baseline models with their quantized counterparts, measuring accuracy drop versus efficiency gains.
  • Identified optimal trade-offs between model accuracy, size, and real-time mobile inference performance.
